# Cradle boilerplate
Boilerplate for your projects made with [Gulp](http://gulpjs.com/), [Sass](http://sass-lang.com/), [Pug](https://pugjs.org/), [webpack](https://webpack.js.org/).

##Installing
```
git clone ...
git submodule init
git submodule update
npm i
```

##Dev build (don't use it on a production server)
```
gulp
```

##Production build
```
gulp --production
```
